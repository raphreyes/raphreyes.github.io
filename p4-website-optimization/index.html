Website Optimization

How to compare versions

The links below will open an optimized or if selected, an un-optimized version of a pizza website.

When comparing the websites, press the F12 key after the pizza site loads to bring up your browser's console. Be sure the Console data is being displayed, as opposed to Element or Network data.

While the console is displayed, scroll the website up and down, and adjust the pizza size slider. The console will output how many milliseconds the page is using to display data.

The original un-optimized page will be close to in the range of 70-90 (or more) milliseconds per frame for scrolling, and somewhere over 400 miiliseconds to change pizza size.

The optimized site should display times of about 0.9 to 5 milliseconds for scrolling, and about 2 milliseconds for changing pizza size.
